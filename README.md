# Stepper Motor Project

Stepper motor control project using **Embedded C** and **Proteus**.

## Files
- `main.c` : Source code  
- `GccApplication12.hex` : Compiled file  
- `proteus.pdsprj.zip` : Proteus simulation  
- `Stepper Motor Report.pdf` : Project report  
- `Stepper Motor.mp4` : Demo video  

## Run
1. Open Proteus project  
2. Load HEX file  
3. Run simulation
